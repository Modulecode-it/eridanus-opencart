# Индекс Задач

Реестр всех задач проекта. Поддерживай актуальным: при создании/завершении задачи обновляй строку.

## Легенда статусов

- `в работе` — задача активна.
- `завершён` — все 4 документа созданы, код вмержен/подготовлен.
- `код подготовлен` — код написан, ждёт ручных шагов/деплоя.
- `заблокирован` — задача остановлена с указанием причины.
- `отменён` — задача отменена.
