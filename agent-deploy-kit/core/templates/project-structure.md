## Документация Задач

Новые нетривиальные задачи документируются с одним общим task-id в четырёх папках. Основной текст — на русском, технические идентификаторы без перевода. См. `CORE.md`.

```text
.agent/
|-- PROJECT_STRUCTURE.md  ← этот файл
|-- STACK.md              ← правила и команды стека
|-- specs/                ← требования
|-- analysis/             ← исследование
|-- implementation/       ← журналы реализации
`-- reports/              ← итоговые отчёты
```
