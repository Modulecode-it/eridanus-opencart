# Итоговые Отчёты

Здесь хранятся итоговые отчёты по задачам.

## Именование

- `<task-id>-final-report.md`, например `1.1-final-report.md`.
- Один `<task-id>` во всех четырёх папках: `specs/`, `analysis/`, `implementation/`, `reports/`.
- Скелет нового файла — `../templates/report-template.md`.

## Рекомендуемые разделы

- Сводка
- Изменённые Файлы
- Проверка (выполненная и невыполненная)
- Известные Пробелы
- Следующие Шаги

## Правила

- Создавайте или обновляйте итоговый отчёт после каждой нетривиальной задачи.
- Включайте ссылки/пути к соответствующей спецификации, анализу и журналу реализации.
- Включайте изменённые файлы, выполненную проверку, невыполненную проверку и известные пробелы.
- Дополняйте полезную историю вместо удаления существующей документации.
- Основной текст, заголовки и пояснения — на русском; технические идентификаторы без перевода.
