## Зафиксированные Ключевые Выводы

- Browser checkout является стандартным OpenCart 3 accordion checkout, управляемым inline AJAX в `catalog/view/theme/default/template/checkout/checkout.twig`.
- Shipping calculation управляется расширениями через `model_extension_shipping_{code}->getQuote($address)`.
- Payment availability управляется расширениями через `model_extension_payment_{code}->getMethod($address, $total)`.
- Order creation происходит в `catalog/controller/checkout/confirm.php` до того, как выбранный payment controller подтверждает order history.
- CDEK, Measoft, `cod_cdek`, `filterit`, `sms_alert`, voucher и Yandex status events являются основными кастомными checkout-related integrations.
