## Hotfix 2026-07-22: подключение карты ПВЗ в SimpleCheckout

### Изменённый файл

**Файл:** `catalog/controller/checkout/simplecheckout.php`

**Причина:** На витрине SimpleCheckout кнопка «Выбрать ПВЗ на карте» вызывала `showModalMea()`, но `catalog/view/javascript/measoftcourier/checkout_script.js` не был подключён к странице.

**Краткое описание:** В начало `index()` добавлено условное подключение ресурсов Measoft при включённом `shipping_measoftcourier_status`:
- `https://home.courierexe.ru/js/measoft_map.js?v4.3.11`
- `/catalog/view/javascript/measoftcourier/checkout_script.js?v4.3.11`
- `/catalog/view/javascript/measoftcourier/style.css`

**Возможные побочные эффекты:** Страница checkout теперь зависит от доступности внешнего JS `home.courierexe.ru` только когда модуль Measoft включён. При повторном применении OCMOD одинаковые URL должны схлопнуться в `Document` по ключу.

### Проверка
- Подтверждено поиском, что `showModalMea` объявлена в `catalog/view/javascript/measoftcourier/checkout_script.js`.
- Подтверждено, что в `catalog/controller/checkout/simplecheckout.php` до правки не было подключения `checkout_script.js`.
- `git diff` показывает только добавление трёх ресурсов Measoft в SimpleCheckout.
- `php -l` не выполнен: PHP CLI в локальном PowerShell не найден (`php` не распознан как команда).
