# ADR-001 Checkout Architecture

## Статус
Draft

## Контекст
Планируется изменение стандартного checkout OpenCart.

## Решение
Все изменения checkout должны выполняться через существующую архитектуру проекта.
Предпочитать изменение исходных контроллеров/шаблонов перед generated modification.

## Последствия
Упрощается сопровождение и обновление проекта.
