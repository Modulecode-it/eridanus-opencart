## 2. Что нужно сделать вручную в админке OpenCart

**Файл для загрузки:** `D:\work\modulecode\eridanus-opencart\measoft_oc3.ocmod.zip`

### Шаг 1. Загрузить OCMOD-архив

1. Зайти в админку OpenCart (`/admin/`).
2. Меню слева: **Модули / Расширения → Установщик** (Extensions → Installer).
3. Нажать кнопку **«Загрузить»** (Upload) → выбрать файл `measoft_oc3.ocmod.zip`.
4. Дождаться зелёной надписи «Успешно».
5. В списке установленных расширений найти **«MeaSoft»** → нажать зелёную кнопку **«Установить»** (➕), если ещё не установлен.

> Если OpenCart пишет «Каталог загрузки недоступен» — нужно создать папку `system/storage/upload/` с правами на запись (обычно уже создана).

### Шаг 2. Применить модификатор (обновить modifications)

1. Меню: **Модули / Расширения → Модификаторы** (Extensions → Modifications).
2. В списке найти модификатор **«MeaSoft»** (code: `measoftcourier`, version 4.3.12).
3. Убедиться, что статус — Включён (зелёный переключатель).
4. Нажать синюю кнопку **«Обновить»** (↻) в правом верхнем углу — это применит все патчи.
5. Дождаться сообщения об успехе.

> После этого шага в папке `system/storage/modification/` появятся файлы с применёнными патчами (admin/controller/sale/order.php, catalog/controller/checkout/checkout.php и т.д.).

### Шаг 3. Дать права администратору

1. Меню: **Система → Пользователи → Группы пользователей** (System → Users → User Groups).
2. Нажать «Редактировать» у группы **Administrator**.
3. В блоке **«Доступ к расширениям»** (Access Permission) найти и отметить:
   - `extension/shipping/measoftcourier`
   - `extension/shipping/measoftcouriershipping`
4. В блоке **«Изменение расширений»** (Modify Permission) — то же самое.
5. Нажать «Сохранить».

### Шаг 4. Активировать основной модуль Measoft

1. Меню: **Модули / Расширения → Доставка** (Extensions → Shipping).
2. В выпадающем списке «Тип расширения» выбрать **«Доставка»** (если ещё не выбрано).
3. Найти **«Measoft»** → нажать кнопку «Установить» (зелёный ➕), если не установлен → затем **«Редактировать»** (карандаш).
4. При первом открытии модуль автоматически создаст колонки в БД:
   ```sql
   ALTER TABLE oc_order ADD pvz_id VARCHAR(20);
   ALTER TABLE oc_order ADD pvz_name TEXT;
   ```

### Шаг 5. Заполнить настройки на тестовый аккаунт

В форме настроек Measoft заполнить:

| Поле | Значение (тестовый аккаунт) |
|---|---|
| **Логин** | `login` |
| **Пароль** | `pass` |
| **Код курьерской службы (extra)** | `8` |
| **Код клиента** | Нажать кнопку **«Получить Код клиента автоматически»** (заполнится сам) |
| **Город отправитель** | Например: `Москва` |

Дополнительно можно настроить:
- **Заголовок ПВЗ / Курьера** — как способ доставки будет называться на витрине (например «Доставка курьером», «Самовывоз из ПВЗ»).
- **Вес по умолчанию (кг)** — `0.1` (если у товаров не задан вес).
- **Размеры карты** — `650 × 755` (по умолчанию).
- **Коэффициент стоимости** — `1` (умножение тарифа Measoft; `1.5` = наценка 50%).
- **Добавочная стоимость** — `0` (фиксированная наценка в рублях).
- **Префикс номера заказа** — например `OC-` (необязательно).
- **Привязка оплат** — выбрать, какие способы оплаты соответствуют CASH (наличные при получении), CARD (картой при получении), NO (предоплата). Если у вас онлайн-оплата — все способы в `NO`.

В самом низу: **Статус → Включено**.
Нажать **«Сохранить»** (дискета в правом верхнем углу).

### Шаг 6. Активировать подмодуль ПВЗ-доставки (опционально)

Если нужна доставка до пункта выдачи (ПВЗ):

1. Меню: **Модули / Расширения → Доставка**.
2. Найти **«Measoft Courier Shipping»** → установить ➕ → редактировать.
3. В поле **«Сервисы доставки»** (`services_delivery`) выбрать нужные режимы из списка (тянутся из API Measoft).
4. **Статус → Включено** → Сохранить.

### Шаг 7. Проверить на витрине

1. Открыть витрину магазина.
2. Добавить товар в корзину → перейти к оформлению заказа.
3. На шаге «Способ доставки» должны появиться варианты:
   - «Measoft курьер» — с рассчитанной стоимостью.
   - «Measoft ПВЗ» — с кнопкой **«Выбрать пункт выдачи»**.
4. Нажать «Выбрать пункт выдачи» → должна открыться **карта** с ПВЗ.
5. Выбрать ПВЗ → стоимость пересчитается → кнопка подтверждения разблокируется.
6. Завершить оформление заказа.

### Шаг 8. Закоммитить сгенерированные модификации (!)

**Это критически важно** из-за особенности проекта: `system/storage/modification/` отслеживается git.

После шага 2 (Обновление модификаторов) в `system/storage/modification/` появились файлы. Их нужно закоммитить:

## 3. Как зарегистрироваться в Measoft для рабочего аккаунта

Сейчас мы используем **тестовый аккаунт** (`login/pass/extra=8`). Для боевого магазина нужно:

### 3.1 Выбрать курьерскую службу
Measoft — это платформа, через которую работают десятки курьерских служб РФ/СНГ. Выберите службу доставки в вашем регионе, которая работает на платформе Measoft.

### 3.2 Заключить договор
- Связаться со службой доставки (или с самой Measoft как с оператором).
- Контакты Measoft:
  - Телефон: **8 (800) 707 90 35**
  - Email: **support@courierexe.ru**
  - Сайт: [courierexe.ru](https://courierexe.ru/) / [measoft.ru](https://measoft.ru/)
  - Wiki-документация: [wiki.courierexe.ru](https://wiki.courierexe.ru/)
- Заключить договор на оказание услуг доставки.

### 3.3 Получить реквизиты
Курьерская служба создаст вам карточку клиента и сообщит:
- **Логин и пароль** для личного кабинета (создаются в карточке клиента → вкладка «Прочее» → «Доступ к сайту»).
- **Extra code** — уникальный идентификатор курьерской службы на платформе Measoft (у каждой службы свой).

### 3.4 Сменить пароль
При первом входе в личный кабинет `https://home.courierexe.ru/{extra}/` сменить временный пароль.

### 3.5 Перенастроить модуль
В админке OpenCart → Measoft → заменить тестовые `login/pass/8` на рабочие реквизиты → нажать «Получить Код клиента автоматически» → Сохранить.

## 5. Возможные проблемы и решения

### Проблема 1: «Каталог загрузки недоступен» в Installer
**Решение:** Создать папку `system/storage/upload/` с правами на запись для веб-сервера:
```bash
mkdir -p system/storage/upload
chmod 775 system/storage/upload
```

### Проблема 2: Карта ПВЗ не загружается на витрине
**Причины:**
- Домен `home.courierexe.ru` недоступен с сервера магазина (файрвол, геоблок).
- В шаблоне оформления заказа отсутствует точка внедрения OCMOD.

**Диагностика:**
```bash
curl -I https://home.courierexe.ru/js/measoft_map.js
```
Если код не 200 — проблема в доступности.

### Проблема 3: Способы доставки Measoft не появляются на витрине
**Причины:**
- Статус модуля = Выключен.
- Не задан город отправитель.
- Гео-зона не включает регион покупателя.
- Не получен client_code (пустое поле).

**Решение:** Проверить все обязательные поля в настройках модуля.

### Проблема 4: Стоимость доставки = 0 или ошибка расчёта
**Причины:**
- Неверные `login`/`pass`/`extra`.
- Не получен `client_code`.
- В API Measoft нет тарифа для маршрута (город отправителя → город получателя).
- Вес корзины = 0 и не задан вес по умолчанию.

### Проблема 5: После deploy модуль снова не работает
**Причина:** Не закоммичены `system/storage/modification/`.

**Решение:** После применения OCMOD-патча обязательно:
```bash
git add system/storage/modification/
git commit -m "chore(measoft): commit modifications"
```
