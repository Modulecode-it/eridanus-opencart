# Спецификация: Установка модуля доставки Measoft 4.3.12

**Задача:** 1.4-measoft-install
**Дата:** 2026-07-18
**Статус:** в работе

## 1. Цель

Полноценная установка модуля интеграции с платформой Measoft (courierexe.ru) для расчёта стоимости и оформления доставки курьером и до ПВЗ в OpenCart 3.0.3.2.

Модуль обеспечивает:
- Расчёт стоимости курьерской доставки до двери (`measoftcourier`)
- Расчёт стоимости доставки до ПВЗ с интерактивной картой выбора пункта (`measoftcouriershipping`)
- Создание заказов в Measoft из карточки заказа в админке
- Запрос статусов доставки
- Сохранение выбранного ПВЗ (`pvz_id`, `pvz_name`) в заказе

### Важный вывод по версиям
Установленный код **новее** оригинального пакета — содержит кастомные патчи:
1. **Конвертация веса** (`getWeight()`): если `config_weight_class_id == 2` (граммы), вес делится на 1000 → конвертация в кг для Measoft API.
2. **Paytype в калькуляторе**: `fetchPaytype()` определяет тип оплаты (CASH/CARD/NO) по коду платёжного метода и передаёт в XML калькулятора Measoft.
3. **Масса в калькуляторе курьера**: `mass => $weight < 1 ? 1 : $weight` — минимальная масса 1 кг для курьерской доставки.

**Решение:** НЕ перезаписывать 4 файла с кастомными патчами. Переписать 11 файлов с CRLF-only различием. OCMOD-патч применить отдельно через Installer.

## 3. Регистрация в Measoft и получение настроек

### 3.1 Что такое Measoft
Measoft (courierexe.ru) — SaaS-платформа диспетчеризации курьерских служб. Схема:
```
[Интернет-магазин] → API Measoft → [Конкретная курьерская служба]
```
Measoft — не сама служба доставки, а диспетчер, через который работают курьерские службы.

### 3.2 Путь A: Тестовый аккаунт (бесплатно, прямо сейчас)

| Параметр | Значение |
|---|---|
| URL ЛК | `https://home.courierexe.ru/8` |
| Логин | `login` |
| Пароль | `pass` |
| Extra | `8` |

Это общедоступная тестовая среда. **Не вводить реальные данные клиентов.**

### 3.3 Путь B: Рабочий аккаунт (для реального магазина)

1. **Связаться с курьерской службой** (работающей на платформе Measoft) или с Measoft напрямую.
   - Телефон Measoft: **8 (800) 707 90 35**
   - Email: **support@courierexe.ru**
   - Сайт: [courierexe.ru](https://courierexe.ru/)
   - Wiki: [wiki.courierexe.ru](https://wiki.courierexe.ru/)
2. **Заключить договор** на оказание услуг доставки.
3. **Получить реквизиты:**
   - Логин и пароль для ЛК (создаётся курьерской службой в карточке клиента → вкладка «Прочее» → «Доступ к сайту»).
   - Extra code — идентификатор курьерской службы на платформе (у каждой службы свой).
4. **Сменить временный пароль** при первом входе в ЛК.
5. **Войти в ЛК:** `https://home.courierexe.ru/{extra}/`

## 6. Схема работы модуля

### Курьерская доставка (`measoftcourier.standard`)
```
checkout → getQuote() → Measoft::calculatorRequest() → XML → home.courierexe.ru/api/ → расчёт стоимости
```

### Доставка до ПВЗ (`measoftcouriershipping.standard`)
```
checkout → карта measoft_map.js → выбор ПВЗ → AJAX quote → calculatorRequest → сохранение pvz_id/pvz_name
```

### Создание заказа (из админки)
```
order_form → AJAX → admin/model → Measoft::orderRequest() → XML → home.courierexe.ru/api/ → статус
```

## 8. SQL-изменения

Метод `install()` контроллера `measoftcourier.php` при активации модуля создаёт в таблице `oc_order`:
```sql
ALTER TABLE `oc_order` ADD `pvz_id` VARCHAR(20);
ALTER TABLE `oc_order` ADD `pvz_name` TEXT;
```

Метода `uninstall()` для удаления колонок нет.
