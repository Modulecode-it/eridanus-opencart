## Key Findings Logged

- Browser checkout is standard OpenCart 3 accordion checkout driven by inline AJAX in `catalog/view/theme/default/template/checkout/checkout.twig`.
- Shipping calculation is extension-driven through `model_extension_shipping_{code}->getQuote($address)`.
- Payment availability is extension-driven through `model_extension_payment_{code}->getMethod($address, $total)`.
- Order creation happens in `catalog/controller/checkout/confirm.php` before the selected payment controller confirms order history.
- CDEK, Measoft, `cod_cdek`, `filterit`, `sms_alert`, voucher, and Yandex status events are the main custom checkout-related integrations.
